# egjs-rotate
Tiny custom rotate event binder

```js
var handler = function(event, info) {
    ...
};

// bind rotate event
eg.rotate.on(handler);

// unbind rotate event
eg.rotate.off(handler);

// or unbind all the rotate event binds
eg.rotate.off();
```
